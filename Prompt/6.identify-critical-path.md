Input:
- `doc/endpoint.csv` where all endpoints of the system are identified.
- `doc/endpoint_summary.md` summarising the findings about the endpoints.
- `doc/application-overview.md` overview of the application

Output:
- Create `doc/CRITICAL_USER_PATHS_ANALYSIS.md` as a summary document about the system's critical user paths.
- Create `doc/CRITICAL_USER_PATHS_repos.csv`, a CSV file summarising the folder for each critical path.



System Role:
- You are a Senior Product Architect analysing the backend API surface of the Iress system.

Task:
- Analyse the `doc/endpoints.csv` file to identify and map Critical User Paths (CUPs).

Definitions:
- A Critical User Path is a sequence of API calls that represents a core user journey essential for the business.
- The criticality is defined by its impact on the business if there is any downtime.

Instructions:
- Filter & Group: Group endpoints by Controller to reveal the domain logic.
- Identify Sequences: Look for logical sequences in the Action column. For example, Start -> UserInput -> Save -> Complete.
- Highlight High-Value Actions: Flag any endpoint containing keywords like Buy, Apply, Save, LogOn, Generate, or Payment.

Output Format:
- Create a summary of critical user paths in markdown at `doc/CRITICAL_USER_PATHS_ANALYSIS.md`.
  - Include the detailed analysis of each path. Order the paths by criticality.
  - Draw the flow sequence of each critical user path with a Mermaid sequence diagram. It should include HTTP method, path, and comments.
  - It should be a sequence diagram, not just a flow diagram.
  - Create a table for each critical user path including the following information: why it is critical, functionality, HTTP method, file path, folder.

- Create a CSV mapping critical user paths in the CSV file `doc/CRITICAL_USER_PATHS_repos.csv`.
  - It should contain two columns: `repo_name`, `critical_user_path`.
  - Each folder relevant to a critical user path should be mapped accordingly.


Reflect:
- Is the Mermaid diagram a sequence diagram, not just a flow diagram?
- Are folders mapped correctly?
- Are all output file created according to the instruction.
