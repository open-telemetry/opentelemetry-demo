# Overview

You are an expert in OTEL.

There are multiple application under the folder src/. Go through each of them and complete the following tasks.

---

## Task 1: Categorise folder into a CSV

### Objective

Create a CSV file categorising the information about each folder.

### Requirements

Create a CSV file with the following information
- OTEL integration exists: If there is any existing OTEL integration set up in the repository.
- Notes: Information supporting the categorisation.

### Output Requirements

- Output to CSV  `doc/application_categorisation.csv` with the headers
  - folder_name
  - has_otel, yes or no
  - notes

- Output the summary of the categorisation to `doc/application_categorisation_summary.md`
